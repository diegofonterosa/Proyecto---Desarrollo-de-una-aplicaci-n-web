# GamerZone - Tienda online de periféricos gaming

Proyecto de desarrollo web realizado como parte del módulo de **Desarrollo de Aplicaciones Web** en el ciclo formativo de **2.º ASIR** (Administración de Sistemas Informáticos en Red) en **Prometeo FP by thePower Education**.

---

## Descripción

GamerZone es una tienda online estática de periféricos y accesorios gaming. El proyecto consiste en tres páginas web interconectadas con un diseño responsive, desarrolladas únicamente con HTML, CSS y JavaScript vanilla, sin usar ningún framework ni librería externa.

## Páginas

- **index.html** — Página principal con sección hero, categorías de productos y productos destacados
- **producto.html** — Página de detalle de producto
- **contacto.html** — Formulario de contacto

## Tecnologías utilizadas

- HTML5
- CSS3 (diseño responsive con Flexbox y Grid)
- JavaScript (vanilla, sin frameworks)

## Estructura del proyecto

```
├── index.html
├── producto.html
├── contacto.html
├── style.css
├── script.js
├── favicon.ico
└── img/
```
